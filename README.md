在跑项目时经常要用到IP 但是又懒得每次都切换，市面上的隧道代理价格高昂，而短效IP的性价比很高，一个IP只需几分钱甚至几厘。
所以让deepseek开发了一套代理池中间层！本工具旨在将持续时间仅有1分钟至60分钟不等的短效IP转变为固定IP供其他工具使用，形成代理池服务器，部署一次即可永久使用。

功能特点
仅支持socks5协议监听
根据API动态获取ip 且具有代理保护机制 仅有请求时才提取 避免多扣费
自动代理检测验证是否可用 如果不可用自动切换
提供web前端可以看到代理的存活时间和下次切换时间
config.ini里进行配置 均有说明

切换模式可以选择：1.每个请求都重新提取ip 2.根据上一次的时间检测是否切换
